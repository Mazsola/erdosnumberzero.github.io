I'm an embedded software engineer working on high-performance image processing applications on a many-core processor architecture, using parallel programming methods. I primarily code in C POSIX, but have also done projects in OpenCL and using a Dataflow programming model.

I organise the Paris Maker Girls meetup, which focuses on creating a community of women and girls interested in the Maker Movement. I facilitate workshops on electronics, Arduino projects, and soft circuits.

I studied Electrical Engineering at UNSW, specialising in image and signal processing.

In my free time, I like to sketch, watch arthouse films, travel, and hang out with my dog. Check out my work on Github (transfer in progress!) or follow me on Twitter.
